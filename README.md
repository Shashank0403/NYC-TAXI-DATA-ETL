## NYC Taxi Data ETL on AWS
This repository contains an end‑to‑end ETL pipeline that ingests NYC Taxi trip data into AWS, stores it in a data lake, transforms it using AWS Glue, and exposes it for analytics using Amazon Athena, all orchestrated with AWS Step Functions and monitored via CloudWatch
